RealTimeGame
============

A real time multiplayer game application, made on node.js and socket.io (framework: sails.js)

Installation
------------

::

    $ npm install
    
To Run
~~~~~~~

::

    $ sails lift
    
Preview on http://localhost:1337

Routes
~~~~~~
1. /auth/login & /auth/register
2. / (homepage)
3. /game/play
